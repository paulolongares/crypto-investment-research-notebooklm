Descrição do Projeto

Este repositório documenta um projeto de pesquisa desenvolvido com o NotebookLM sobre como investir com segurança em criptomoedas. O caderno foi construído utilizando fontes confiáveis, análise fundamental e métricas on-chain para estruturar conhecimento e gerar relatórios claros e contextualizados. O objetivo é organizar dados, métodos e explicações de forma transparente para apoiar decisões baseadas em informação e não em especulação.
